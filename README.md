# AI Sentiment Analyzer

This is a beginner-friendly AI mini project developed using Python.
It analyzes a sentence entered by the user and predicts whether the sentiment is **positive**, **negative**, or **neutral**.

## Features

* Text sentiment detection
* Positive / Negative / Neutral classification
* Keyword-based NLP logic
* Easy to run and understand

## Tech Used

* Python

## How to Run

```bash id="jl9gpf"
python sentiment_analyzer.py
```
